Install Redmine Bridge

One-time setup — takes 2 minutes
1
Download the extension folder from your GitHub repo — it's in the extension/ folder. Or download the ZIP from the release page.
2
Open Chrome and go to chrome://extensions
3
Turn on Developer mode (toggle in top-right corner)
4
Click Load unpacked → select the extension/ folder
5
Click the 🧩 puzzle icon in Chrome toolbar → find Redmine Bridge → click it → enter your Redmine URL and API key → Save
6
Come back here — the sidebar will show Connected ✓ and you can pull tasks from Redmine!
💡 API Key: In Redmine → top-right avatar → My Account → scroll to bottom → API Access Key → Show
